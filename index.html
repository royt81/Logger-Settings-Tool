Logger Settings Tool

Standalone editor for the Logger extension's data lists. Edit entries below, then generate JSON to paste into the Confluence page. This tool is not part of the Logger extension itself.

Data source
Log Code

Optional tag inserted into the copied log line, e.g. "RC" for DD.MM.YYYY_RC_INITIALS: message. Leave blank to copy just DD.MM.YYYY_INITIALS: message — matches logCode in a-lists.js.

Output

Click "Generate JSON" any time after editing, then copy it into the Confluence page.
